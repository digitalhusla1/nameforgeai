🤖 Google Gemini API

Google's advanced generative AI model for creative business name generation.

📖 API Example 
curl "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=YOUR_API_KEY" \
  -H 'Content-Type: application/json' \
  -X POST \
  -d '{
    "contents": [
      {
        "parts": [
          {
            "text": "Generate creative business names for a coffee shop"
          }
        ]
      }
    ]
  }'

🧠 Anthropic Claude API

Anthropic's Claude AI for sophisticated and contextual business name suggestions.

📖 API Example 
curl https://api.anthropic.com/v1/messages \
  -H "x-api-key: YOUR_API_KEY" \
  -H "content-type: application/json" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "claude-3-5-sonnet-20241022",
    "max_tokens": 1000,
    "messages": [
      {
        "role": "user",
        "content": "Generate creative business names for a coffee shop"
      }
    ]
  }'

🦙 Replicate API

Access to various open-source language models via Replicate platform.

📖 API Example 
curl -s -X POST \
  -H "Authorization: Token YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "version": "meta/llama-2-7b-chat",
    "input": {
      "prompt": "Generate creative business names for a coffee shop"
    }
  }' \
  https://api.replicate.com/v1/predictions
